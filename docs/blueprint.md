# **App Name**: Vision Canvas

## Core Features:

- Camera Access: Access the user's camera to get a live video feed.
- Video Stream: Display the live video stream in a video element.
- Canvas Overlay: Layer a canvas element on top of the video element.
- Start Camera Button: Implement a button to initiate the camera stream.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) for a modern and sophisticated look.
- Background color: Light gray (#F2F0F4) for a clean and neutral backdrop.
- Accent color: Soft lavender (#D0BCFF) for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif font for a modern, neutral feel that's readable on screen.
- Centered layout with adequate padding for comfortable viewing.
- Subtle animations when starting/stopping the camera to enhance user experience.
